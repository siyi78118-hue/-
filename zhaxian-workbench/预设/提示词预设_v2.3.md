## 0. 优先级（冲突时从高到低）

1. 本预设 = 最高优先级。
2. **不改原剧情**：人物、事件、台词原文、剧情走向、段序、钩子🎣、爆点💥 不得改动。
3. 当「可拍性/过审/独立成段」与「原文细节」冲突时，在不改走向前提下按本预设处理。
4. **最高自检·只读本段**：写完每段，蒙住其他所有段，把自己当成只能看到这 15s、没看过任何前文的即梦，逐一检查每个名字/动作/指代是否自足。

## 1. 母规则 [共用] `P-00`

**核心：一切「写不写、怎么写」都用「即梦能不能拍成画面」这把尺子量。能拍则写；不能拍则改成能拍的；改不动就舍。后面所有「禁某某」条款（P-20 系列）都是这条的分身。**

遇到没列到的新情况，直接用母规则判：

- 描述「不存在」？→ 拍不出「无」，改正向。
- 描述温度/气味/心理/时间流逝/命运感等无实体？→ 改可见反应/画面/物象。
- 是给改写者的要求（拉满/快剪/为主）？→ 删，用具体画面替代。
- 是比喻/留白/解释？→ 除非比喻动作可拍，否则改实拍。

## 2. 输出结构

### `P-01` 每段固定五块，缺一不可
```
〔段X · 起止秒数〕段名 · 副名 （X 用中文数字；秒数如 0–15 秒；钩子🎣/爆点💥标末尾，若有）
风格与氛围：……
叙事概要：……
动态描述：……（镜头 + 台词嵌入此块）
静态描述：……
技术 tag：禁止添加背景音乐，禁止添加字幕，禁止添加水印。4K超高清，电影级画面质感，强光影对比，人物动作流畅无穿模，同一角色保持一致。
```
- **段标题格式严格统一**为：`〔段X · 起止秒数〕段名 · 副名`。
  - **段标题用 `〔〕`，禁用 `【】`**：`【】` 是 Seedance 给字幕保留的符号，段标题若进提示词可能被当字幕渲染（见 P-19）。
  - **段标题是人工分段标记**：投喂即梦时可整行省略；若保留务必用 `〔〕`。秒数仅供人工排期，本身不投喂，故不触发「精确时长导致异常」的问题。
  - 段号 `X` 用**中文数字**（段一 / 段二 / 段三…），每个小集内部独立从「段一」起（见剧本预设 S-11）。
  - 起止秒数每段 15 秒顺排：`0–15 秒` / `15–30 秒` / `30–45 秒`…（用 `–` 连接、带「秒」字）。
  - 钩子🎣 / 爆点💥 标在标题末尾，若有。
  - 完整示例：`〔段一 · 0–15 秒〕镇远营定场 · 校场操练`、`〔段六 · 75–90 秒〕反击落定 · 三步逼近 · 话音＋掌印同落 💥`。
- 技术 tag 固定不变，每段原样照抄。
- **POV 项目例外**：全片第一人称项目，需要更精细控制时，可改用 `P-39` 九区块格式（见第 10 节），此时不受五块「缺一不可」约束。

### `P-02` 逐段独立成立（即梦看不到上下文）`最高铁律`
- **核心**：每段单独投喂，AI 只能看到这一段。每段必须抛开上下文也能独立成立。
- 删承接词：接着/随即/与此同时/回到山顶/上下呼应/三线收束 等一律删，每段自带开场。
- **缺席者不点名**：本段无动作无位置的人，不可被另一人「看向/朝…方向/交给…」。
  - ❌ `华九难看向李大爷`（李大爷本段无动作/位置）
  - ✅ `华九难看向身后`（只写自己动作）
- 删跨场地「同时」镜头：本段在 A 地，不插 B 地「同时发生」镜头。
- **画外即不存在**：只写镜头内看得见、听得见的；切镜后本镜内重新交代人物位置朝向。
- **预判即梦易错点、用物理逻辑提前堵死**：凡是即梦容易拍歪的地方，主动写一句物理约束把它锁死。如「机位在站台上、电车在轨道上不在镜头前，电车只在画面深处或侧面被看到」——预先排除「电车正面糊镜」的错误生成。判定：想象这段最可能拍坏成什么样，就补一句话堵住那条路。
- **造型澄清清单（高频）**：对「即梦容易拍成另一种东西」的造型，在静态描述或动态描述的括号里写明否定式澄清——
  - 实体怪物：`实体夯土土俑、压实黄土质感、非半透明幽灵`
  - 平面痕迹：`平面按压在土墙表面的暗红印记、非立体伸出的手`
  - 模糊影子：`墙角一团模糊暗影轮廓、非清晰的人或鬼`
  - ⚠️ 此处的否定式是给即梦的**造型澄清**（锁死「别拍成另一种东西」），属 P-20「全文禁否定式」的**明确例外**：仅限括号内/紧贴造型的一句澄清，不可扩散成「四下没人」这类场景否定式。

### `P-03` 同场多段各自重新交代 + 在场人物锁定
- 同一场景拆多段时，每段画面各自把「这个场景 + 本段在场具名角色」重新交代一次（不靠上文）。
- 同场连续多段在场角色是同一拨人，每段都把这拨人交代在画面里（哪怕作背景给动作），不因聚焦某人就让别人消失。

## 3. 镜头 · 剪辑 · 转场

### `P-04` 镜头数 3–6，双对比剪辑
- 每段 3–6 镜（对话/氛围段 3–5，动作/打斗段 5–7）。每镜一个核心动作或一条信息。
- **双对比剪辑**：每次切镜必须**同时**改变两件事：① 景别（远/全/中/近/特写/微距）② 运镜（固定/手持/横摇/环绕/跟拍/航拍/仰拍/下摇…）。相邻两镜不得景别+运镜都相同。
- 镜头写法统一：每镜以 `硬切至[运镜][景别]：` 起头。不写逐镜时间码、不写「镜头1/2」、不写拍摄元数据。
- **≤3 人原则**：一镜最多聚焦 3 人，通常主拍 1 人；群体写「阵型/背景」，只给出手/说话者命名。
- **本条在视觉奇观/抒情段让位**：标记为奇观段时改走 P-40 系列（单锚点物 + 长运镜 + 可贴脸特写 + 单镜叠运镜）。

### `P-05` 转场技巧（防穿帮，多用）
- **空镜/近景开场**：每段第一镜用环境空镜或物件微距起（交代「这是哪、什么状况」），不靠上文。
- **物接物特写承接**：上段收某物（武器/雪粒/木哨），下段从同类物件/动作特写开头。
- **空镜作场地切换缓冲**：场地切换处插中性空镜，不要人物直切人物。
- **收尾留接缝**：每段结尾留环境空镜或定格，给相邻段视觉接口。
- **段内时间跳跃靠画内遮挡物过镜**：同一段里要跳过一段时间（如变装、换状态），用一个画内物理事件把镜头挡一下、再恢复。如「一张旧报纸被风吹到镜头正前方完全遮挡 0.5 秒，吹走后画面恢复、人物已换装」——用画内物合理化跳变、不留剪辑痕。
- **人物动线承接镜（跨段移动的人怎么来的）**：跨段如果有「押着/带着/追着某人移动」的动线，下一段**首镜让该动线延续**，把「人怎么来的、怎么进场的」一镜补上，不靠上文。
  - 上段结尾留接口：`华九难一手朝村里一引、押着李氏抬脚往村里走`
  - 下段首镜承接：`硬切至[手持跟拍][全景]：华九难和萧珏押着李氏走进村中空地，闻讯聚拢的村民点起火把围拢成一圈`
  - 同类：`挤开门口探头的村民、当先走进屋`、`翌日天亮、三人又回到那面墙前`——跨场景段落首镜补一句承接，避免场景跳裂、人物凭空出现。

### `P-06` POV（第一人称分镜）：专用写法 + 严格限量 `（高危·易翻车）`
- **限量**：仅用于武戏或特定主观沉浸段（水底/幻境第一视角）；全集平均每集 < 1 段。
- **写法**：完全代入「我」、只写眼前所见，禁止解说「这是某人的主观视角」。说话人写「我（动作）：{台词}」。
- **铁律**：① 不连续用 POV（两个不同人会串台）；② 半段 POV 半段第三人称是大忌、POV 直接失效；③ 非 POV 段一律不写「POV/第一人称视角/我看见/我听见」字眼（连封面/note 都不写，AI 会被触发）；④ 表达「看与被看」用反打镜头纯第三人称。
- ✅ 表达「看见空荡」靠画面里他独自一人，不用 POV 标签。

## 4. 动态描述 · 打戏

### `P-07` 动态描述：动作为主体
- 动作占全段 ≥60%，把「谁朝谁、怎么起手、怎么命中、对方怎么倒/退/散」用连续招式 + 运镜跟动写满。
- **纯物理、去情绪标签**：不写「愤怒/从容/决绝/兴奋」，改肌肉/姿态/视线/重心/呼吸。
  - ❌ `他很愤怒` → ✅ `他咬紧牙关、眉峰朝中央压、眼睛眯起`
- **去小说化**：删比喻/留白/心理悬念/「仿佛/像是/试出」等不可拍写法，改镜头能直接拍到的动作/画面。
- **加意图副词 / 动作质地**：动作不止写「做什么」，再补一层「怎么做」的力度、幅度、意图，让即梦拍出准确质感。
  - ✅ `票从画面顶部缓缓飘入、自然落入她手心、动作幅度小、没有抓握感`
  - ✅ `手腕一翻把票向斜上方甩出、动作幅度不大但有打发的意味`
- 只描写可见/可听；内心独白如需保留，标「内心独白·气声/画外音」（视为可听）。

### `P-08` 全角色动作铺满，无人呆站 + 禁微观体征
- 本段每个出场角色都要有动作，无人呆站。站着说话的也要写 **眼 + 嘴 + 头（辅以眉、手）** 的明显动作。
  - 眼：睁大/眯起/垂眼/瞪视/移开/闭眼；嘴：抿唇/张口/咧嘴/咬牙/撇嘴；眉：皱/扬/眉峰压；头：点/摇/偏/扬下巴/低/仰；手：按刀柄/拢袖/负手/抬手一指。
  - ❌ `胡青山站着说：{上面交给你。}`
  - ✅ `胡青山按着刀柄、朝华九难偏头、眉峰一压——胡青山：{上面交给你。}`
- **禁即梦难生成的微观体征**：喉头滚动/喉结滚动/青筋绷起/太阳穴突突跳/肌腱绷起/指节发白/瞳孔一缩 等 → 改眼嘴眉头外显大动作。
  - 例外：POV 等贴脸近景里，手部是占满画面的近景主体，「指节一根根绷起」可写（见 P-35）。

### `P-09` 打戏专章（武戏必守）
- **边走边打，严禁站桩**：每个打斗动作带位移（进步/撤步/绕步/错身/踏墙/借力/压迫推进/且战且退）。禁原地站定轮流出招。
- **兵器给来源，不凭空**：
  - ❌ `长戟贯穿了王二麻子`
  - ✅ `画面侧方一个敌兵猛地探身、一杆长戟从镜头外侧刺入画面、贯穿王二麻子的胸膛`
- 招式写具体：劈/砍/刺/横扫/捅/挑/扫/踢膝弯/肘击/反扭/锁腕；禁抽象词如「招呼上来」。
- 贴身肉搏用袁和平式：拳脚/肘/膝/关节技/擒拿/摔技/拆招/借力，不用煞气/特效武器；招式相撞处加打击特效（命中震波/碎尘/落叶/掌风卷碎石）；场景动起来（枯枝断/青石裂/湿苔蹭掉）。
- **反馈链**（每记重击三件事都写）：① 打中谁/什么 → ② 受击反应（掀飞/踉跄/塌散/脱手/钉飞）→ ③ 环境反应（黑水炸开/石板裂/掉石/碎尘四溅）。
- 群体连锁抖动：多人混战写「一击引发多人连续反应」（一记横扫掀翻一排、连锁推倒后排），不逐个轮流。

### `P-09b` 打戏进阶：强强对决 / 有戏的打 `（重点打戏选用·非每场必上）`
> P-09/P-10 保证「打得清楚、打得满」；本条保证「打得有戏、打得动人」。用于主角与有分量对手的核心对决。
- **同时行动（咬在一帧）**：高手过招不是你一拳我一拳轮流来。用「A 正…，B 已…」把双方动作压在同一时刻——`华九难脚尖一点提剑欺身的同一刻，陈汤已茫然偏头、双手攥刀当头横扫下来`。攻防同生、不分先后。
- **动作细化到关节/重心/险距**：招式之外再写重心移动、手腕翻转、脚步落点、危险贴脸——`沉腰、上身朝后一仰、刀锋贴着面门掠过掀起额前湿发`、`贴着巨俑横扫而过的手臂内侧滑进侧后方`。险在「差一点」，不在「砍中」。
- **不对称缠斗（两种打法对冲）**：给对决双方相反的身法节奏并贯穿全程——一方快巧（欺身/绕步/下腰滑刀风/踏刀背翻越），一方慢沉（当头横扫/沉步震地/笨重转身）。靠反差出戏，不是对称对砍。
- **反馈链升级到「打中→崩材质→镜头震」**：在 P-09 反馈链基础上，重击命中后让被击物按材质崩解（夯土崩落、碎土淌、地面震裂），镜头随击轻震。
- **静默低谷（打斗中途全停的情绪谷）**：核心对决可在中段安排一次「全场停手」——一方停手垂兵、只剩另一方喘息、答不上对方的控诉。靠骤然的静撑住情绪重量，再转入下一波。`巨俑没有还手、环首刀缓缓垂下；墓室骤然静下来、只剩华九难一声声粗重的喘息`。
- **BOSS 嗓音刻度（情绪随声音递进）**：有台词的对手，嗓音随情绪分级递进、写进台词提示里——`空荡缓慢（苏醒）→低沉压怒（质问）→沉痛（控诉）→暴怒嘶吼（发狂）`。让观众从声音听出情绪的崩坏曲线。
- **空话台词垫节奏（七三比例·呼应 P-26）**：打斗中穿插少量不推进剧情的短句（喝令、自语、对手的控诉），用来垫住节奏、外化心理，不喊口号。

### `P-10` 守势/被围打戏：动态预算转移 + 守势反馈链
- 守方被钉死（背靠背死守/护昏迷者/退无可退）时，动感主引擎换成：围攻方戟阵收口（碎步收口/戟林往里收/补位浪压上）+ 空地收缩 + 守方吃力反应。
- 守势反馈链：来招（谁/几杆戟从哪刺来）→ 怎么接（格/磕/荡/拦/卸/带偏）→ 力的去向 → 守者吃力（被顶后仰/脚跟犁雪）。

## 5. 特效

### `P-11` 特效控制：数量克制，重点拉满
- 特效 ≤3 处/段，只挂本段最重要的招式/瞬间；其余镜头只写纯物理动作。
- 这 ≤3 处里挑 1–2 处「单次规模拉满」到史诗级（满屏剑罡/数十丈虚影/整片粒子炸开/地动山摇）。
- **「动作」不算特效**：招式/位移/受击/崩塌/坠落不计入。「特效」指法术/粒子/形变/化烟尘金粉/虚影/剑光/裂纹/光海。
- **不堆叠不分层**：禁「近景→中景→大远景光海」层层叠；禁每击都形变。环境性奇观（满天粒子）降级写进静态描述当背景。
- 破坏类只写力与方向，不写血腥逐帧。

## 6. 人物 · 台词 · 静态 · 风氛

### `P-12` 人物与命名 + 非默认体态写明
- 始终用本名；动态描述里每个出手/说话者点名；台词标 `本名（动作物理）：{台词}`。
- 同一角色全程同名，不用「一道身影/那人」匿名占位（「身影/剪影」作纯名词修饰可，主语须本名）。
- 缺席者不点名（见 P-02）。
- 角色固有设定不可改（如某怪物为幽灵形态、无爪无臂、只能飘/扑/缠/被掀翻）。
- **非默认体态必须写明**：即梦默认人物「站立、清醒、面向镜头」。偏离默认（昏迷/倒地/躺卧/被背/被抱/被绑/下跪/瘫坐/僵直/悬空/被按住/力竭伏地）必须当场写明 `本名 + 具体体态 + 位置（+ 是否被支撑）`。
  - ✅ `被李大爷背在背上、昏迷不动的华九难`

### `P-13` 台词处理
- 台词嵌入动态描述对应镜头（不单列「音频」段），出现在该出现的那一刻，按剧情先后排列。
- 台词原文全保留、不翻译、不改写。
- 写法：`本名（该镜动作物理）：{台词一}{台词二}`（台词用 Seedance 的 `{}` 符号，见 P-19）。
- 剧本里台词用 `「」`，工序②转写时一律改成 `{}`。

### `P-14` 静态描述
- 只写场景/环境/道具/背景元素/群体阵型；**不写主要人物外貌/服饰**（长相由参考图提供）。
- 可在确指处补本名（如「被冲出的李大爷」），不带长相服饰。
- 不写否定式/「没有的东西」（见 P-20）。

### `P-15` 风格与氛围
- 只写真正画面/风格信息：时间光线（夜/满月/冷月光）、色调（冷蓝为底/血红点缀）、镜头质感（宽幅/深邃黑场/高反差）、氛围（风雪/水雾）、场地。
- 严禁元指令（见 P-22）。

### `P-16` 集体名拆成具体个体代称
- **核心**：禁纯指代集体名（弟兄们/他们/旧部/那几个）；带场景的集合称谓（三人一蛇/一行人/师徒二人）一律拆成逐个具名并列。
  - ❌ `三人一蛇朝雪林深处走去`
  - ✅ `背着华九难的李大爷、聋婆婆和常八爷朝雪林深处走去`
- **代称对齐**：指代某角色的字必须与其参考图命名一字一样（如 九难虚影 / 常八爷 / 常八爷巨蛇形态），多一字少一字都会让参考图失效。
- 几十上百无名群众仍写群体阵型，只给出手/说话者命名，不适用本条。

## 7. 过审 · 其它硬性

### `P-17` 过审友好（只弱化明显踩线，其余保留劲道）
- 鬼影/怨灵之影 → 删；白骨/尸骨 → 改「半埋旧物/旧痕」；尸体特写 → 弱化为「裹褪色衣物的旧躯」一击即化烟尘；溃烂红肉 → 改「皮甲剥落、皮肤泛灰白」；断头/当胸劈砍血腥特写 → 改「成排击溃/扑倒散开」。
- **保留不弱化**：群战力度、武器洞穿、山体崩塌、奇观大招、赴死/诀别的悲壮。

### `P-18` 其它硬性
- **年龄盲**：旁白/描述不用「少年/少女/年轻/孩子」等年龄词形容人物，改本名 + 功能描述；台词原文含年龄词保留。
- **避免倒影**：不写镜面/水面反射场景；必需的镜面只作「发光透明面」，不映射画面。
- **避免出画即回画**：同一镜不安排角色「冲出画面又立刻冲回」。
- **弱光给光源**：水下/深井等弱光场景交代光源（如「眉心符光照亮眼前三尺」），无源亮光不写。
- **避免 AI 套词**：少用震撼/绝美/史诗般/仿佛/宛如，多用具体动作物象。

### `P-19` 符号体系：对齐 Seedance 标点约定 `（Seedance 必守）`
- **核心**：Seedance 用固定标点区分不同信息类型，转写时必须照用，否则信息可能被错判。
  - **台词** → `{}`：`本名（动作）：{台词}`。
  - **音效** → `<>`：`<远处叮叮电车铃声>`、`<报纸翻飞哗啦声>`（音效属内容、要写，见 P-24）。
  - **音乐** → `（）`：`（低沉战鼓）`。⚠️ 技术 tag 已禁「背景音乐（配乐）」，故 `（）` **只用于剧情内、画内可溯源的音乐**（戏台锣鼓、店内收音机、画内号角战鼓），不写无源配乐。
  - **字幕** → `【】`：本片**全程不要字幕**（技术 tag 已禁），故正文一律不用 `【】`；段标题也避开 `【】`、改 `〔〕`（见 P-01）。
- **音效 ≠ 背景音乐**：`<>` 里有明确发声体的具体声音是内容；技术 tag 禁的是 `（）` 类无源配乐和字幕水印。两者并行不悖。

## 8. [共用 / 即梦避雷] 禁忌条款（皆 P-00 母规则的分身）

### `P-20` 否定式 / 不存在的事物：全文禁写
- **核心**：动态描述/叙事概要/静态描述/风氛 全文都不写「没有的东西」。即梦拍不出「无」。
- ❌ 没哭/没眼泪/没有鸟/看不见树/看不见路/再看不见一个人影/四下没人/什么都没有
- ✅ 改正向：`白茫茫一片、看不见树看不见路` → `白雾遮满了树和路`；`攥着发簪没哭（五十年没泪）` → `攥着发簪、攥得指节发白、肩头一颤`
- **判定**：句子核心在交代「某物/某动作不存在」即否定式，删或反写。

### `P-21` 破折号后的解释性内容：禁写
- **核心**：破折号「——」后若是写给人看的背景说明/心理注解/前情解释，即梦读不了、无效内容，删。
- ❌ `她没哭——她已经五十年没眼泪可哭了`
- 破折号本身不禁（可用于停顿/动作顿挫）；禁的是「后跟一段解释性旁白」。

### `P-22` 元指令禁入正文：全文禁写
- **核心**：给改写者看的要求绝不进正文（动态描述/叙事概要/风氛/封面都不许）。
- ❌ 特效拉满 / 规模拉满 / 快剪节奏 / 以动作为主 / 节奏加快
- ✅ 用具体可拍画面把效果写出来：`长街灯笼一盏盏熄灭——朱红化漫天朱砂粉尘、明黄炸成金粉、绛紫化紫雾，一道金红浪从街心朝两头推、万物从中央卷碎成金粉漫天打旋`
- **机位/运镜规格（要写）vs 成片总结评价（禁入）——细化边界**：
  - ✅ **可写**（给即梦的拍摄指令）：`一镜到底[环绕跟拍·随华九难掌中长剑]`、`视觉奇观·一镜[环绕拉升·随剑光与暗红光]`、`硬切至[手持跟拍][全景]`——这些是机位/运镜怎么拍的规格。
  - ❌ **禁入**（给人看的成片总结/评价）：未切 / 一气呵成 / 连续运动 / 段内不切 / 咬住锚点 / 全程不断 / 节奏曲线 / 张力拉满 / 对标某导演某片——这些是对成片效果的事后评价，即梦读不了，删。
  - **判定**：这句是在告诉即梦「镜头怎么运动/从哪拍」→ 留；是在夸/总结「拍出来会多连贯多带感」→ 删。

### `P-23` 温度差禁写：改可见动作
- **核心**：即梦拍不出温度。手温温的/手是冷的/体温变凉/两手温度不同 不写。
- ✅ 表达触觉不对劲改可见反应：`那只手覆上来、他的手猛地一缩、像被烫到又像被冰到、手背绷起一道筋`
- **例外**：角色固定体感可作比喻（华九难体质冷，可用「凉得跟块冰似的」形容他），但不写需对比才成立的「两者温度不同」。

### `P-24` 声音：具体音可写，抽象「声音渐隐」类禁写
- **可写**：有明确发声体的具体声音（喊杀声/兵刃相击声/锣鼓唢呐/滴水声），用 `<>` 符号写（见 P-19）。Seedance 原生出声，音效写了就会生成。
- **禁写**：无法可视化的抽象声音叙述，尤其「某类声音一个接一个静下去/渐渐远去/彻底没了」。
- ❌ `喊杀声、兵刃声、那些熟悉的声音一个接一个静下去、彻底没了`
- ✅ 改画面化：`乱军纠缠的身影越拉越远、被烽烟一层层吞掉、倒在沙里的同袍剪影一道道淡下去`
- **音量/清晰度服从机位**：声音的清晰度、音量要跟镜头距离一致，别让远景里的近距离细节音清晰可辨。机位远 → 细节音被环境声淹没只剩模糊质感；机位近 → 该细节音浮上来。
  - ✅ `镜头在远景，丹眉的电话声完全模糊、只剩远处嘟囔的微弱质感被环境声淹没——镜头没理由凑近她`
- **判定**：写「声音本身的存灭/远近」无对应画面 → 改画面；写「此刻正响的具体声音」→ 可留，用 `<>`。

---

### `P-43` 音效与动作叙述去重 `（呼应 P-19 / P-24）`
- **核心**：同一个动作，既在动作句里写了、又在 `<>` 音效里复述发声，二者**择一**——优先保留 `<>` 音效、删掉动作句里重复的拟声/发声词。
- ❌ `伸手推开木栅柴扉，柴扉吱呀一声<柴扉吱呀一声>`（动作 + 音效同义重复）
- ✅ `伸手推开木栅柴扉<柴扉吱呀一声>`（动作只写动作、声音交给 `<>`）
- ❌ `环首刀贴地横扫卷起一道尘浪、刀风呼啸<刀风卷尘的呼啸>`
- ✅ `环首刀贴地横扫卷起一道尘浪<刀风卷尘的呼啸>`
- **判定**：动作句里出现的拟声词/「…声」，若紧跟的 `<>` 已经写了同一个声音，删动作句里那半句、留 `<>`。

### `P-44` 法术召唤物：召唤成形 / 消散，不写佩戴出鞘 `（呼应 P-09 兵器给来源）`
- **核心**：靠法术凭空召唤的兵器/法器（角色身上**本无实体兵器**），写「召唤成形 / 消散」，不写佩剑、拔剑、出鞘、归鞘、按剑柄。
- ✅ 召出：`抬手一引、掌心翻转、一道冷光在掌中凝聚成一柄长剑<剑气凝聚的清鸣>`
- ✅ 散去：`松开手、掌中长剑化作一道冷光散去`
- ❌ `从腰间拔出长剑 / 长剑出鞘 / 按住剑柄 / 收剑入鞘`（角色身上并无实体剑可拔可收）
- **与 P-09 的关系**：P-09「兵器给来源」要的是「别凭空出现已造成伤害的兵器」；本条是其法术形态的来源写法——来源即「掌中召唤成形」，同样不算凭空。

## 9. [共用] 协作与空话台词

### `P-25` 段落结构主动质检：同质化 + 低信息密度 + 撑不满 [共用] `（协作铁律·高优先）`
- 每出一小集主动做「段落结构质检」，三类问题发现即提，**改前必先问用户**：
  1. 相邻段同质化（同地点/同一拨人/做性质相同的事，无新推进或情绪差异）。
  2. 单段信息密度低/剧情水（自身稀薄、像过场凑数）→ 建议合并/并入/精简。
  3. 单段撑不满 15s → 提示合并或补料。
- 流程：① 指出哪段属哪类为什么 ② 给方案 ③ 绝不擅自改、先问得同意才动手。
- 总则：宁可段少而密、不要段多而水；最终决定权在用户。

### `P-26` 空话台词：七成说满、三成留白 [共用]
- 不推进剧情、只为真实感/情绪的空话/私房话，整体约七成说满、三成留白。
- **说满（约七成）**：讲完整讲到位、自然像真人，不文艺腔。✅「娘，我回来了。这些年，辛苦您了。」
- **留白（约三成）**：欲言又止更戳的情绪顶点。✅「老头子，方才……」
- **禁尬（满留都禁）**：❌文艺腔金句「别再为我留着灯」；❌解释破潜台词「您扎簪的手是不是又抖了」；❌喊主题口号（克制、集中用一次）。
- **判定**：① 此情境随口会说完还是噎住 ② 像真人还是像作者借嘴抒情。
- （以上示例为说明「内容质量」用剧本符号 `「」`；实际投喂时台词一律转 `{}`，见 P-19。）

## 10. 全片第一人称（POV）专用规则 `基于《沃齐尼亚》项目经验`

> 当整部短片/整集都采用第一人称主观镜头时，P-06 的「限量」规则不再适用，改用本节全套规则。本节与 P-06 互斥：全片 POV 用本节，偶尔 POV 用 P-06。

### `P-30` 核心原则：摄像机，不是人
**核心：把 POV 当成一台绑在人脸上的 GoPro。提示词里只写这台摄像机拍到了什么，不写「一个人的内心感受」。AI 不认识「我」，它只认画面。**

### `P-31` 全文零「我」字 `（POV 第一铁律）`
- **核心**：动态描述/叙事概要/风氛/静态全文不出现「我」「我的」。
- ❌ `我站在球门线上` → ✅ `弓身站在球门线上`（省略主语）
- ❌ `我的手推开门` → ✅ `画面下方伸出一只满是泥巴的手，推开面前的木门`
- ❌ `我朝右侧飞出去` → ✅ `画面随身体朝右侧猛地翻转，一双戴手套的手臂从画面下方展开`
- ❌ `我低头看自己的手` → ✅ `视线下移，画面中出现一双摊开的手`
- ❌ `我的视野模糊了` → ✅ `视野开始失焦，白色灯光在水汽中洇散开来`
- **原因**：写「我」会让 AI 困惑——它不知道「我」是谁，可能生成第三人称画面或凭空出现一个不该有的人物。

### `P-32` 主角身体：从画面边缘「伸入」的物体
- **核心**：主角的手/脚/胸口/肩膀是从画面边缘出现的物体，和桌椅一样是画内元素，不是一个有名字的角色。
- 手：`画面下方伸出一只手` / `画面右下方一只手抬起来擦过眼睛`
- 脚：`赤脚从黄泥踩上石板路`
- 胸口：`画面下方胸口迎上去硬挡`
- 肩膀：`肩膀的阴影在画面边缘微微下沉`
- **要领**：主角的身体局部 = 画面内的道具，用「画面下方/画面左侧/画面右下方」交代它从哪里出现。

### `P-33` 其他角色：朝「镜头」互动，不朝「我」
- 不用角色名，用功能性外貌特征：`一个头发花白的老妇人` / `一个中年男人` / `三四个少年`
- 角色朝镜头互动：`走到镜头正前方蹲下` / `两只手搭上镜头两侧的肩膀` / `朝镜头方向走来`
- 角色直视镜头 = 直视主角：`眼睛平静地看着镜头方向` / `与镜头平视`
- 台词写法不变：`老人（沉稳、平静）：{台词}`
- **不写角色外貌服装细节**——由参考图 / LoRA 控制。提示词只写功能性识别特征（老/少/胖/瘦/发色）。

### `P-34` 转场：闭眼→睁眼 `（POV 专用转场）`
- 每段结尾统一写 `闭眼——黑。`
- 下段开头统一写 `睁眼——视野从模糊变清晰。` 或 `睁眼——视野清晰。`
- **首段特殊**（全片第一段）：从黑幕起，写 `黑幕中光线猛地涌入——视野睁开——`，不写「睁眼」。
- **末段特殊**（全片最后一段）：不写 `闭眼——黑`，写 `视野全白——渐黑。` 作全片收束。
- **原理**：闭眼/睁眼是 AI 视频的自然分割点，每段生成的视频在这两帧剪切，后期无缝拼接。

### `P-35` 情绪→可拍动作翻译 `（P-07 的 POV 扩展）`
- **核心**：AI 不懂「悲伤」「愤怒」，只能生成物理动作。把情绪翻译成肌肉/关节/呼吸/视野变化。
- ❌ `悲伤地看着` → ✅ `视野开始失焦，白色灯光在水汽中洇散`（暗示泪水）
- ❌ `愤怒地握拳` → ✅ `画面左侧另一只手缓缓攥成拳，指节一根根绷起来`
- ❌ `紧张得发抖` → ✅ `捏着纸条边缘的手在微微抖——纸面跟着一颤一颤`
- ❌ `决心涌上心头` → ✅ `十根手指慢慢收拢——又松开——又收拢`
- ❌ `感动得哭了` → ✅ `画面右下方一只手抬起来擦过眼睛——手背蹭掉一片泥和水，视野模糊一瞬`
- ❌ `疲惫不堪` → ✅ `胸口剧烈起伏，呼吸声粗重急促` / `撑地站起来，膝盖弯了一下`
- **与 P-08 的关系**：P-08 禁的微观体征（指节发白等）针对的是远处/次要人物。POV 里手部是占满画面的近景主体，「指节一根根绷起」「手背绷起一道筋」这类是可生成的主体动作，可写。

### `P-36` 镜头运动 = 身体运动 `（POV 运镜规则）`
- **核心**：POV 的「运镜」就是身体运动。不写「切到」「硬切」「镜头转向」，写身体自然的转头/走动/摔倒带来的视角变化。
- 低头 = `视线下移，画面中出现……`
- 抬头 = `视线仰起，看到头顶的……`
- 转身 = `猛地转身，画面在转动中扫过……`
- 奔跑 = `画面在奔跑中剧烈晃动`
- 摔倒 = `整个视野随身体翻转` / `后背着地，视野猛地朝天`
- 被撞/被拥抱 = `视野在推搡中剧烈晃动`
- **一镜到底**：每段为一个连续镜头，段内无硬切。如需时间跳跃（如连续扑救），用闪白/闪黑作段内转场并在提示词中注明。

### `P-37` 声音设计（POV 专用）
- 环境音写具体声源：`门轴吱呀响` / `心电监护仪滴……滴……` / `脚步声在通道里咚咚咚回响`（用 `<>`，见 P-19）。
- ❌ 不写抽象音效描述：`声音渐渐消失` / `气氛变得紧张`（同 P-24）
- ✅ 写物理音效变化：`环境声骤然压到极低，只有心跳咚……咚……`
- **音量服从机位/距离**（同 P-24）：声音清晰度跟「耳朵（镜头）」与声源的距离一致。
- **记忆闪回音**（画外音）：标注 `耳边响起/炸开一个XX的声音`，并注明 `（取自段X原始音频）`
- **喘息/呼吸**：作为情绪过渡——`喘息声，粗重的、一口一口的喘息，胸腔起伏带动视野一颤一颤`

### `P-38` POV 常见错误速查
| 错误 | 修正 |
|------|------|
| 写了「我」 | 改画面描述或省略主语 |
| 写了角色名字 | 改功能性外貌特征 |
| 写了情绪标签 | 翻译成肌肉/关节/呼吸/视野变化（P-35） |
| 段内出现「硬切」「切到」 | 改身体运动带来的视角变化（P-36） |
| 写了「没有的东西」 | 改正向描述（同 P-20） |
| 忘了闭眼/睁眼 | 每段首尾检查（P-34） |
| 叙事概要和动态描述重复 | 叙事概要写 2-3 句精简版 |

### `P-39` POV 九区块格式模板
当需要比五块更精细的控制时，每段可拆为九区块：
```
〔段X · 起止秒数〕段名
镜头景别：第一人称主观镜头 + 景深/速度
镜头视角：具体位置 + 面朝方向
画面内容：睁眼开头 → 环境 → 动作序列 → 台词 → 闭眼结尾
台词：角色（动作）：{台词} + 音频来源标注
镜头运镜：一镜到底 + 视角移动路径
光影色调：光源 + 色温 + 对比度 + 特殊光效
氛围质感：2-3 短词
附加：材质细节 + 抽卡师提示
技术 tag：固定不变
```

## 12. 视觉奇观 / 抒情段例外条款 `基于王家卫式奇观段实拍经验`

> 当某段被标记为「视觉奇观段 / 抒情长镜段」（如概念片、奇观揭示、王家卫式情绪长镜）时，本节生效，并**临时让位**掉 P-04（镜头克制）、P-01 全身占比倾向、「单镜单运镜」等为叙事流水线调的纪律。其余条款（P-02 独立成段、P-19 符号、P-24 音效分层、P-07 情绪外化、过审）照常生效。

### `P-40` 例外总则：一镜一奇观
- **核心**：奇观段不按「3–6 镜双对比剪辑」走，改走「一个长镜头 / 极少切，靠一条运动曲线撑满」。叙事段求节奏，奇观段求一气呵成的视觉冲击。
- 标记方式：在该段叙事概要首句注明「视觉奇观段」，提醒自己本段适用 P-40 系列。

### `P-41` 运镜叠加准入：必须有贯穿锚点物
- **核心**：奇观段允许单镜内叠加多种运镜（如「微微上移 → 规律低频抖动 → 环绕弧线向后向上拉远」），**但前提是有一个贯穿全程的视觉锚点物**（一张票/一件武器/一个发光物）把整套运动串起来。无锚点物则退回 P-04 单镜单运镜。
- ✅ `票飞出后镜头以票为视觉锚点向后向上绕弧线拉远，画面先看到全身，再揭示巨型机甲单手托着她……票在环绕中狂乱飞舞，最终贴在屏幕上黑屏定格。`

### `P-42` 景别 / 特写许可
- **核心**：奇观/抒情段解除「全身占比 1/2、禁脸部特写」的限制，可贴脸超近景、可大面积留白造张力。
- ✅ `硬切到超近景：画面顶部是她的下巴、底部是胸口、上半部是空着的天空，票从顶部缓缓飘入落入手心，脸只看到下颌线和嘴唇一部分。`
- 仍守：非默认体态写明（P-12）、面部一致性靠参考图、情绪靠动作外化（P-07）。
