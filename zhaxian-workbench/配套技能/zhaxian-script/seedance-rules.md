# Seedance 2.0 剧本优化规则（蒸馏自《Doubao Seedance 2.0 系列提示词指南》）

> ⚠️ 兜底文件。现在工作台每步「发给 AI」的指令里都自带【本步预设】（用户的《剧本预设》/《提示词预设》/特供预设），
> **以指令里那段预设为准**。仅当指令没带预设时，才退回本文件作通用判据。
>
> 原始指南全文见同目录 `seedance-guide-raw.txt`。

## 0. 工具定位
产出是**优化后的剧本范本**（更适合 Seedance 出提示词），不是裸提示词。每个片段(beat)=一段固定 **15 秒**视频；超过 15 秒的连续内容必须拆成多段。

## 1. 动作描述（指南 P7 第3节）
四件套：**肢体细化 + 程度量化 + 低缓连续 + 前后承接**。
- **肢体细化+量化**：具体到手/腿/头/肩背，补幅度(微微/轻轻/大幅)、速度(缓慢/快速/顺势)、力度(用力/轻柔/颤抖)。例：`缓慢抬手、快速转头、用力蹬地、微微低头`。
- **优先低缓连续小动作**（硬约束）：尽量规避狂奔、大跳、剧烈翻滚等高爆发大动态（易崩坏穿模）。把"飞身狂奔/凌空翻滚"降格为"衣袂轻摆、缓步踏前、抬手凝出微光"。快节奏打斗靠**分段拼接+短动作衔接**，不靠单镜写大动作。
- **过渡衔接**：写明前后动作惯性承接，例 `借转身惯性顺势抬手`。
- **按人物给移动方式**：仙法御风/踏空、步行、乘骑、法器(剑/舟/伞)、瞬移。仙法/瞬移也要写成低缓可控（瞬移=身形虚化→数尺外凝实的连续过渡，不写炸裂闪现）。

## 2. 情绪外化（指南 P8 情绪表）—— 用身体细节替代抽象词
| 情绪 | 外化为动作与细节 |
|---|---|
| 悲伤 | 低头、肩膀微微颤抖、眼眶泛红、手指无意识攥紧衣角、泪水在眼眶打转但没落下 |
| 喜悦 | 嘴角抑制不住上扬、眉眼舒展、脚步变轻快、下意识哼起小曲、忍不住原地转个圈 |
| 紧张/焦虑 | 频繁看手表、手指不停敲击桌面、呼吸急促、眼神闪躲、无意识啃咬指甲 |
| 愤怒 | 双拳紧握、下颌线紧绷、胸口剧烈起伏、眼神如刀、从牙缝里挤出话语 |
| 释然 | 长长舒一口气、紧绷的肩膀完全放松、淡淡微笑、抬头望向远方 |

仙侠扩展（仿此范式自创等价的、低缓可被镜头看见的身体外化）：决绝/悲悯/孤傲/震惊/隐忍/杀意/惊惧/释怀。
例·决绝：抿紧嘴唇、缓缓闭眼再睁开、指节因握紧泛白、转身时步伐不再回头。

## 3. 运镜（指南 P5 运镜写法 + P7 分镜时序 + P9 文/武戏）
- **景别**：远景→全景→中景→近景→特写→大特写。
- **运镜术语**直接照搬：推/拉/摇/移/跟/环绕/俯/仰、手持跟拍、无人机俯瞰、固定机位、缓慢推近、平稳横移。
- **一个镜头只用 1 种运镜**（硬约束）：别同时推拉摇移，会抖会崩。要换运镜就切下一个镜头。
- **文戏=连续长镜头（视频延长）**：长对话/情绪递进/单一路径移动，沉浸一镜到底。**武戏=分段拼接**：追逐/打斗/转折，分别生成片段再剪辑。
- **每个镜头四要素**（缺一不可）：①运镜或切换 ②主体动作与表情 ③位置/空间变化 ④音频信息。
- **不写精确秒数**：模型对 0–3 秒这类支持不稳定。用"镜头1/镜头2/镜头3"按事件先后排。

## 4. 台词与符号（指南 P10–P11）
| 类型 | 符号 | 例 |
|---|---|---|
| 台词 | `{}` | `{你好，世界}`；小语种标语种：`用日语说道{こんにちは}` |
| 音效 | `<>` | `<远处传来狗叫声>` |
| 音乐 | `（）` | `（背景中播放着快节奏的摇滚乐）` |
| 字幕 | `【】` | `【第一章：启程】` |

- **台词语言统一、绝不中英混用**（专有名词除外）。
- 适合配音口播：常用字，避开生僻字/多音字/特殊符号（生僻字易读错，可换同音常用字）。台词长度按该语言语速。

## 5. 约束词 / 画质 / 风格（指南 P5、P8、P11–P15）—— 段尾收口
- **画质**：`高清，细节丰富，电影质感，色彩自然，光影柔和`。
- **风格**（防漂移，务必显式写）：`3D国漫CG仙侠风格`、`烟雨江湖电影感，冷调低饱和，电影胶片质感`。
- **无字幕**：`保持无字幕，避免生成任何文字或字幕`（横屏比竖屏少字幕）。
- **无 Logo/水印**：`不要生成Logo`、`不要生成水印`。
- **防双胞胎/ID漂移**：清晰定义每个主体并标注对应参考图；段尾加全局约束 `视频全程禁止出现外形、着装、配饰完全一致的人物，禁止双胞胎分身，同一画面仅保留单个对应人物`。人物参考用大头照+全身照，别用三视图/多视图。

## 6. 进阶公式与分镜时序（指南 P5–P7）
公式：**精准主体 + 动作细节 + 场景环境 + 光影色调 + 镜头运镜 + 视觉风格 + 画质 + 约束条件**。
口诀：先锁"谁在干什么"→再交代"在哪、什么氛围"→再说"怎么拍"→最后用风格/画质/约束收紧。
复杂片段写成**时间轴化分镜**：每镜 = 谁 + 在哪 + 做什么 + 镜头怎么动 + 音频。
