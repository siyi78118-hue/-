        谪仙剧本工作台 · 使用说明（换电脑必读）
════════════════════════════════════════════════════════

这是什么？
  一个本地小工具，帮你把「按《剧本预设》写好的剧本」一步步加工成
  「可直接投喂即梦 / Doubao Seedance 的五块提示词」。
  ★ 不联网、不调任何 API、不花钱。
  ★ 真正干优化活的「大脑」是 AI（Claude）——也就是和你对话的我。
    这个网页只是把流程理顺、把成品上色的「台子」，里面没有藏 AI。

────────────────────────────────────────────────────────
【最简单的用法：双击就能用】
────────────────────────────────────────────────────────
1. 双击打开「谪仙剧本工作台.html」（任何浏览器都行，断网也能开）。

2. 先放预设（只需一次，会随工程保存）：
   点右上角「⚙ 预设」→ 把你的两份预设全文粘进去：
     · 剧本预设（工序① · 用于第①步解析剧本）
     · 提示词预设（工序② · 用于第②~⑥步）
   预设框也支持把 .md / .txt 文件直接拖进去。
   ★ 预设会时时更新——更新后回来把对应框重新粘一遍即可。
   ★ 某一步若有「特供预设」，填到该步的特供框；留空就自动用上面的全局预设。

3. 左边 6 步，从第①步开始：
     ① 录入·解析   把剧本（Word）交给 AI，解析成「小集→段」结构
     ② 动作设计     给每段设计怎么演
     ③ 表情·情绪外化 抽象情绪→能被镜头看见的身体细节
     ④ 台词精修     每句给多个精修方案，任你挑或自己改
     ⑤ 运镜设计     景别+运镜+主体动作+空间变化+音频
     ⑥ 分集出成片   拼成五块提示词，按小集分集导出

4. 第①步「录入·解析」怎么喂剧本（推荐）：
   把你按《剧本预设》写好的【剧本 Word 文档】，连同本步「✨ 发给 AI：解析剧本」
   复制出的指令，一起拖给 Claude 发送 → Claude 按预设解析后写回 →
   点「📥 读取 AI 结果」回填。
   （没用 Word 也行：把纯文本剧本粘到第①步的「剧本原文」框，或把 .txt/.md 拖到拖拽区。）

5. 第②~⑥步：点底部「✨ 发给 AI（本步）」→ 自动复制指令（已自动拼好对应预设）→
   粘给 Claude → Claude 回一段 JSON → 回网页点「📥 读取 AI 结果」粘进去 → 解析并套用。
   （这一来一回，就是"你和 AI 对话"。没有别的接口。）

────────────────────────────────────────────────────────
【换电脑后，怎么把没做完的进度带过去？】★重要
────────────────────────────────────────────────────────
进度（含你放的预设）存在「浏览器里」，不会跟着文件自动搬。所以：
  • 旧电脑：网页顶部点「💾 导出工程」→ 存下一个 .json 文件。
  • 新电脑：打开网页 → 点「📥 导入工程」→ 选那个 .json → 接着做。
（这个 .json 就是你这部剧的全部进度 + 预设，随身带着它即可。）

────────────────────────────────────────────────────────
【强烈推荐：开"桥接"，从此告别复制粘贴】★省事
────────────────────────────────────────────────────────
开了桥接后，整个流程变成：网页点「✨ 发给 AI」→ 回 Claude 说一句「继续」
→ 结果自动填回网页。不用再复制、不用再粘贴、不用再点【读取】。

一次性准备（只做一次）：
  1. 把本文件夹「配套技能\zhaxian-script」整个，放到你项目下的
     .claude\skills\  里面（让 Claude Code 认得这个技能）。
  2. 确认电脑装了 Python（Mac 一般自带；Windows 没有就去 python.org
     下载，安装时勾选「Add Python to PATH」）。

每次使用：
  1. 启动桥接：
       · Windows：双击「启动桥接.bat」
       · Mac：双击「启动桥接.command」（若提示无法打开，右键→打开 一次）
       · 或在本文件夹运行命令： python3 bridge.py
     它会自动打开网页 http://localhost:8765/谪仙剧本工作台.html
     （★ 一定要用这个 localhost 网页，不要再双击 html，否则不能自动联动）
  2. 照常走 6 步。每步点「✨ 发给 AI」——指令会自动投给 Claude，不用你粘。
  3. 回到 Claude Code，说一句「继续」。Claude 处理完，网页几秒后自动填回。
  4. 想连「继续」都省：对 Claude 说  /loop 处理请求  ，它会自动盯着、有
     请求就处理（不想用了就停掉 loop）。
  —— 嫌装这些麻烦？不开桥接也行：双击 html + 手动复制粘贴，功能完全一样。

────────────────────────────────────────────────────────
文件夹里都有啥？
  谪仙剧本工作台.html          ← 主程序（手动用法：双击它）
  启动桥接.bat / .command       ← 一键开桥接（推荐用法：双击它）★
  bridge.py                     ← 桥接服务器本体（被上面的启动器调用）
  ①先看我·使用说明.txt         ← 本文件
  预设\                         ← 两份预设范本（剧本预设 / 提示词预设）
  谪仙工程\ai-out.json          ← 桥接：Claude 写结果到这，网页自动读
  谪仙工程\请求.txt             ← 桥接：网页写请求到这，Claude 自动读
  配套技能\                     ← 桥接用的技能，放进项目 .claude\skills\
