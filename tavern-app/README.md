# 故事酒馆 - 部署指南

## 文件说明
- `index.html` — 主应用
- `manifest.json` — PWA 配置（让浏览器知道这是可安装的应用）
- `sw.js` — 离线缓存支持

## 部署方法（任选其一）

### 方法一：GitHub Pages（推荐，免费）
1. 注册/登录 GitHub (github.com)
2. 新建一个仓库（Repository），名字随意，比如 `tavern`
3. 把这三个文件上传到仓库
4. 进入仓库 → Settings → Pages → Source 选 `main` 分支 → Save
5. 等几分钟后，访问 `https://你的用户名.github.io/tavern/`

### 方法二：Vercel（免费）
1. 注册 vercel.com
2. 新建项目，上传这三个文件
3. 部署后会给你一个 `xxx.vercel.app` 的地址

### 方法三：Netlify（免费）
1. 打开 app.netlify.com
2. 直接把整个文件夹拖拽到页面上
3. 自动部署，给你一个 `xxx.netlify.app` 的地址

## 安装到手机桌面

### iPhone / iPad
1. 用 Safari 打开部署后的网址
2. 点底部的「分享」按钮（方块+箭头图标）
3. 选择「添加到主屏幕」
4. 点「添加」，桌面上就会出现 🏮 图标

### Android
1. 用 Chrome 打开部署后的网址
2. 点右上角菜单（三个点）
3. 选择「添加到主屏幕」或「安装应用」
4. 确认安装

## 使用说明

### 首次使用
1. 打开应用，先去「⚙️ 设置」页
2. 选择 API 类型（OpenAI 兼容 或 Claude）
3. 填入你的中转站地址和 API Key
4. 填入模型名称
5. 点「测试连接」确认可用
6. 保存设置

### 导入角色卡
1. 回到主页，点右下角「＋」按钮
2. 选择角色卡文件：
   - `.json` 文件：TavernAI V1/V2 格式的角色卡
   - `.png` 文件：SillyTavern 导出的带内嵌数据的角色卡图片
3. 导入成功后角色会出现在主页列表

### 聊天
- 点击角色卡开始对话
- 点右上角 📝 可编辑当前对话的系统提示词
- 点右上角 🔄 可重置对话

### 聊天记录
- 所有聊天记录自动保存在浏览器本地
- 在「💬 历史」页面查看和管理
